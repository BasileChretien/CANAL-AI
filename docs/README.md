# docs/ (internal)

This folder holds the project's internal working documents: charter
drafts, team rosters, IRB status, planning notes, and context files
used during Claude sessions.

All files in this folder are **gitignored** by default. The public-facing
description of CANAL-AI lives in the top-level `README.md` and on the
project website (generated from `site/`).

If you are a collaborator and need access to these documents, contact
the project PI directly — they are shared out-of-band.
