本ウェブサイトに関するプライバシーに関するお知らせ

本ページは本ウェブサイトに関する事項のみを対象といたします。パイロット研究において取得・利用・保管・第三者提供される臨床データを規律する科学研究上のプライバシー枠組みとは独立しており、後者については 仕組み および 規制上の位置付け をご参照ください。

要点

  • 本サイトでは、Cookie、トラッカー、アクセス解析、広告のいずれも一切使用しておりません。
  • 本サイトを通じて研究チームが個人情報を取得することはございません。ただし、お問い合わせフォームを通じてご送信いただいた場合を除きます（後述）。
  • 本サイトは GitHub Pages（GitHub, Inc.（米国））上でホスティングされており、不正利用防止の目的で訪問者の IP アドレス等が短期間ログとして記録されます。
  • お問い合わせフォームは、研究チームのメールアドレスがサイト上に露出しないよう、Web3Forms（フォーム送信内容をメールに中継するサービス）を経由して送信されます。
  • フォントはサイト内で自己ホスティングしており、Google Fonts その他の第三者 CDN への通信は発生いたしません。
  • 患者様の情報は本サイト上には一切掲載しておりません。臨床研究プロジェクトは別途 GDPR／個人情報保護法に準拠した枠組みに従い、現在も倫理審査委員会（IRB）およびデータ保護影響評価（DPIA）の審査中でございます。現時点において、患者様のデータの取得・利用・保管・第三者提供は一切行っておりません。

本サイトが取得する情報

本サイトが自動的に取得する情報はございません。アクセス解析スクリプト、Cookie、トラッカーのいずれも設置しておりません。クライアント側で利用する唯一の保存領域は、canal-ai.lang という名称の localStorage キーであり、利用者様が選択された表示言語（en、ja、fr のいずれか）を記憶するためにのみ用いられます。当該キーには個人情報は一切含まれず、ブラウザのサイトデータを消去された場合には削除されます。

お問い合わせフォームをご送信いただいた場合には、入力された項目（氏名、メールアドレス、任意の件名、メッセージ本文）が Web3Forms（後述「本サイトから発生する第三者への通信」をご参照ください）に送信され、同サービスがメールにて研究チームへ中継いたします。当該データは本サイト上には保管されず、データベースや GitHub のリポジトリに保存することもございません。

ホスティングおよびアクセスログ

本サイトは GitHub Pages により配信されております。GitHub, Inc. はセキュリティおよび不正利用防止の目的で、訪問者の IP アドレスを含む短期間のアクセスログを保持しております。詳細は GitHub のプライバシーステートメントをご参照ください。当該ログを研究チームが受領または利用することは一切ございません。

本サイトから発生する第三者への通信

  • 外部リンク（ORCID、LinkedIn、共同研究機関のウェブサイト、SECOM 科学技術振興財団、FedBioMed、GitHub リポジトリ等）をクリックされた場合、ご利用のブラウザが直接当該第三者と通信を行います。これらについては各事業者のプライバシーポリシーが適用されます。
  • お問い合わせフォーム（Web3Forms）。 トップページのお問い合わせフォームをご送信いただいた場合、入力データ（氏名、メールアドレス、任意の件名、メッセージ本文）は HTTPS により Web3Forms へ送信され、同サービスがメールにて研究チームへ中継いたします。研究チームのメールアドレスは Web3Forms の管理画面上で設定されており、本サイト上に掲載されることはございません。Web3Forms は研究チームの委託先（個人情報の取扱いを委託する事業者）として送信内容を処理いたします。詳細は Web3Forms のプライバシーに関するお知らせ および 利用規約 をご参照ください。なお、フォーム送信内容を本サイトまたは当該リポジトリに保管することは一切ございません。Web3Forms のサーバが日本国外に所在し得ることから、本送信は個人情報保護法第28条に定める外国にある第三者への提供に該当する場合がございます。

本サイトに氏名等が掲載されている方々について

本サイトには研究代表者および主任研究者（Charles Dolladille 博士、Basile Chrétien 博士、西田 一貴 博士）ならびに 謝辞 セクションに掲載された関係者の氏名等が記載されております。掲載されているご本人で削除をご希望される方は、お問い合わせ セクションのフォームを通じて研究チームまでご連絡ください。速やかに該当箇所を削除いたします。

利用者様の権利

EU 一般データ保護規則（GDPR）および個人情報の保護に関する法律（個人情報保護法）に基づき、利用者様は、研究チームが利用者様に関する個人情報・保有個人データを取り扱っている場合における当該情報の開示請求、内容の訂正・追加・削除、利用停止等を求める権利を有します。これらの権利は、(a) 謝辞欄に氏名等が掲載されている場合の当該氏名、(b) 研究チームに対して利用者様が発信された通信、(c) お問い合わせフォームを通じてご送信いただいた内容、のそれぞれに及びます。なお、学術研究機関等が学術研究の用に供する目的で取り扱う場合の例外（個人情報保護法第57条第1項）に該当する場合がございますが、その場合であっても研究チームは適正な取扱いに努めるものといたします。権利行使をご希望される場合は、トップページの お問い合わせ フォームを通じて研究チームまでご連絡ください。

本お知らせの変更について
